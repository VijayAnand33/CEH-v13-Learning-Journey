# can-utils

## Overview

can-utils is an open-source collection of Linux utilities for working with the Controller Area Network (CAN) protocol. It allows security professionals to monitor, analyze, generate, record, and replay CAN traffic for testing automotive and industrial communication systems.

---

## Key Features

- Captures CAN traffic.
- Replays recorded CAN messages.
- Monitors CAN communication.
- Generates CAN frames.
- Supports virtual CAN (vcan) interfaces.
- Assists in automotive security testing.

---

## Common Usage

```bash
cansniffer -c vcan0

candump -l vcan0

canplayer -I <log_file>
```

---

## Practical Example

During **Module 18 – IoT and OT Hacking**, can-utils was used to capture CAN traffic generated by the ICSim simulator, record communication into a log file, and replay captured messages to demonstrate a replay attack against the CAN protocol.

---

## Defensive Perspective

Automotive and industrial systems should implement message authentication, secure gateways, and network segmentation to reduce the risk of replay attacks. Monitoring CAN traffic also helps identify unauthorized communication and abnormal device behavior.
