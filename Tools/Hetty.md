## Overview

Hetty is an open-source HTTP toolkit designed for web application security testing. It functions as an interception proxy that allows security professionals to capture, inspect, modify, and replay HTTP requests and responses during authorized penetration tests. Hetty serves as a lightweight alternative to commercial interception proxies while providing essential features for web security analysis.

---

## Key Features

- HTTP proxy interception
- Request and response inspection
- Proxy logging
- Request replay
- Project management
- Manual traffic modification
- MITM testing support
- Modern web dashboard

---

## Common Usage

Typical workflow:

1. Launch Hetty.
2. Create a new project.
3. Configure the browser proxy.
4. Capture HTTP requests.
5. Analyze intercepted traffic.

## CEH Practical Example

During **Module 11 – Session Hijacking**, Hetty was used to intercept HTTP traffic generated by a victim accessing the MovieScope web application. The captured POST request exposed the submitted username and password, demonstrating how unencrypted HTTP communication enables credential theft during session hijacking attacks.

---

## Defensive Perspective

Organizations should require HTTPS for all authenticated communications, encrypt sensitive data in transit, implement secure cookies, and deploy HSTS to prevent attackers from intercepting HTTP requests and stealing user credentials.
